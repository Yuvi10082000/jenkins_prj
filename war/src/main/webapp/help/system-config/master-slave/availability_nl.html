Controleert de beschikbaarheid van een agent node.
Hou deze agent node zoveel mogelijk beschikbaar.
Dit is de standaard instelling. In deze modus zal Jenkins de agent node altijd beschikbaar proberen te houden. Indien Jenkins deze agent node automatisch can starten, dan zal een poging ondernomen worden om deze, bij onbeschikbaarheid, periodisch te herstarten. Jenkins zal de agent node niet onbeschikbaar maken.
Maak de agent node op vraag beschikbaar en onbeschikbaar indien de agent node onbenut is.
In deze mode zal Jenkins, indien mogelijk, de agent node periodisch (her-)starten zolang er jobs zijn die aan de volgende criteria voldoen:
  • Ze staan al minstens voor de opgegeven opstartperiode in de wachtrij.
  • Ze zijn uitvoerbaar op deze agent node
De agent node zal onbeschikbaar gemaakt worden indien:
  • Er geen actieve jobs aanwezig zijn op de agent node
  • De agent node inactief is voor minstens de opgegeven periode
