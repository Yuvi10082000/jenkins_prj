Une description optionnelle de cet agent, lisible par les humains. Cette information est affichée sur l'écran de configuration du projet.

Quand vous avez des agents avec des paramètres différents des autres agents, il peut être utile d'expliquer ces différences à travers ce champ. Par exemple, "Agent Windows" permet aux propriétaires des projets de choisir de toujours construire sur une machine Windows (typiquement, s'ils ont besoin d'un outil spécifique sous Windows).
