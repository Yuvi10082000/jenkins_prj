Si le build nécessite un accès à l'environnement graphique (par exemple pour lancer des tests d'interface swing, etc) cette option doit ê sélectionnée. Si ce n'est pas le cas, Jenkins exécute le build dans un processus 'headless'.

Pour assurer la compatibilté ascendante, cette option est activée par défaut pour les jobs créés avant son introduction dans Jenkins.
