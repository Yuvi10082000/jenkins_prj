This field defines the location of the private local repository if you use this option in your jobs. By default, the private repository will be created under the current workspace (${WORKSPACE}/.repository) but if you wish, you can override this value.

For instance, by using m2repo/${EXECUTOR_NUMBER}, one private local repository per executor will be created, which will use less storage than having a private repository per job while ensuring only one instance of Maven accesses it.

This enables you as well to relocate private local repository to a different storage with more space.

You can specify either a path that is relative to the instance home (JENKINS_HOME for master, BASE for slaves) or an absolute path. You also have access to environment variables provided by Jenkins.
