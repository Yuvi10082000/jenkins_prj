Der Archivierer benutzt von Ant die Klasse org.apache.tools.ant.DirectoryScanner welche standardmäßig symbolische Links nachverfolgt. Wenn als Artefakte z.B. die Datei a.log sowie ein symbolischer Link b.log -> a.log erzeugt werden, dann wird das Muster "*.log" beides archivieren.

Diese Option kann benutzt werden um das Nachverfolgen von symbolischen Links abzuschalten. Falls nicht gesetzt, dann wird das Muster "*.log" nur a.log archivieren aber nicht b.log.
